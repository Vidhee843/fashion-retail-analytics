# Interactive Dashboard

## Overview

Web-based interactive dashboard for exploring fashion product analytics results.

## Features

- 📊 Overview panel with key metrics and visualizations
- 🔍 Interactive cluster explorer with dropdown selection
- 🎯 Live category predictor tool
- 💡 Comprehensive insights panel

## How to Use

1. Open `fashion_dashboard.html` in any modern web browser
2. No installation required - runs entirely in browser
3. Click through tabs to explore different analyses
4. Use dropdowns and buttons for interactivity

## Technology

- HTML5/CSS3/JavaScript
- Chart.js for visualizations
- No backend required - fully client-side
