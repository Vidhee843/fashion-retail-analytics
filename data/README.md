# Data Directory

## Dataset Information

**Name:** Fashion Product Images Dataset  
**Source:** Kaggle  
**URL:** https://www.kaggle.com/datasets/paramaggarwal/fashion-product-images-dataset  
**License:** Database: Open Database, Contents: Database Contents

## Download Instructions

1. Visit the [Kaggle dataset page](https://www.kaggle.com/datasets/paramaggarwal/fashion-product-images-dataset)
2. Download `styles.csv`
3. Place `styles.csv` in the project root directory (or update path in R script)

## File Description

**styles.csv** (11.2 MB)
- 42,409 rows (products)
- 10 columns (attributes)
- Variables: id, gender, masterCategory, subCategory, articleType, baseColour, season, year, usage, productDisplayName

## Note

The actual data file is NOT included in this repository due to size and licensing. Please download directly from Kaggle.
